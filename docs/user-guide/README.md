## ðŸ’¡ Tips for Users

### Best Practices
- Always enter supplier serial number (required)
- Add DTX internal serial for customer-facing devices
- Link vendor invoices as soon as they arrive
- Set warranty dates when known
- Use filters to find devices quickly

### Common Questions
- **Q:** What's the difference between supplier and DTX serial?
  - **A:** Supplier serial = from manufacturer (for warranty). DTX serial = optional internal code.

- **Q:** Can I search by both serials?
  - **A:** Yes! Search works on both fields.

- **Q:** What happens if I don't have vendor invoice yet?
  - **A:** Mark as "Invoice Missing". Update later when invoice arrives.
